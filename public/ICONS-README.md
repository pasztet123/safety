# PWA Icons

Aplikacja wymaga ikon PWA w następujących rozmiarach:
- pwa-192x192.png
- pwa-512x512.png
- apple-touch-icon.png

## Jak wygenerować ikony:

1. Stwórz prostą ikonę (np. w Canva lub Figma):
   - Tło: #171717 (czarny)
   - Symbol bezpieczeństwa (np. kask budowlany, znaczek ostrzegawczy)
   - Akcent czerwony: #EE2E2F

2. Wyeksportuj w następujących rozmiarach:
   - 192x192px jako pwa-192x192.png
   - 512x512px jako pwa-512x512.png
   - 180x180px jako apple-touch-icon.png

3. Umieść w folderze `/public/`

## Alternatywnie:

Użyj generatora online:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Wgraj podstawową ikonę 512x512px i pobierz wygenerowane pliki.

## Tymczasowe rozwiązanie:

Możesz użyć favicon.svg jako podstawy dla wszystkich ikon.
